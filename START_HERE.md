## 💬 WHAT TO DO IF...

### You Get an Error
1. Check browser console (F12)
2. Read the error message carefully
3. Check PROJECT_SUMMARY.md troubleshooting section
4. Restart dev server: `npm run dev`

### You Want to Add a Section
1. Create new component in `src/components/`
2. Import in `src/pages/Home.jsx`
3. Add to the JSX render
4. Style with Tailwind CSS

### You Want to Change Colors
1. Open `tailwind.config.js`
2. Update `primary` and `accent` color values
3. All components automatically update!

### You Want to Connect a Database
1. Update form in `src/components/DemoForm.jsx`
2. Replace fetch URL with your API endpoint
3. Test with console logs
4. Deploy!
